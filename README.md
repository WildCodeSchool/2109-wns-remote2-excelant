# 2109-wns-remote2-excelant

## Contributing

### Installation

-   Clone this repo

-   #### Start
    -   Back-end
        ```
        cd server
        npm i
        npm run start:dev
        ```
    -   Front-end
        ```
        cd web-client
        npm i
        npm start
        ```
