# 2109-wns-remote2-excelant

## Contributing

### Installation

1.  Clone this repo

    ```
    git clone https://github.com/WildCodeSchool/2109-wns-remote2-excelant.git
    cd <app-name>
    ```

2.  #### Start

    -   Back-end
        ```
        cd server
        npm i
        npm run start:dev
        ```
    -   Front-end
        ```
        cd web-client
        npm i
        npm start
        ```
